Chonky-19

Adjustable Projectiles
Implemented projectiles that can be customized (spread, speed, damage, number of projectiles in one shot)
Animations
Animated 3D cat model (idle, standing & shooting, running forwards, running forwards & shooting, running backwards)

duckball

Play as an optimistic-looking beach ball to bounce on peoples' heads and eat trash on a beach because a duck said so. A short 3D game with 2D assets created for a 2-week game jam.

Read Game Design Document

"Play on itch.io" takes you to the main game page. Towards the bottom of the page are my development logs if interested.
Programming and Art Assets
Implemented all of the gameplay mechanics from scratch (movement, sticky items, growing volume, platforms that launch player in direction of normal vector, scoring, time limit, etc) and created 2D assets to fit jam theme
UI and Animations
Implemented basic UI (title screen, dialogue system, score display, timer display) and created simple idle animations for each NPC in game.
Honorable Mention
Received an Honorable Mention out of 395 submissions to the game jam
